# nestimate-ios
It is an app the manages household expenses for mulpiple users
### Features

- User Login and Sign up 
- Create/Edit Home
- Invite user to home
- Create/Edit/Delete expenses for home
- Create/Edit/Delete cateogries of expenses
- Profile
